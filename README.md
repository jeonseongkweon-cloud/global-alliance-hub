# GLOBAL ALLIANCE HUB

공식 메인 대문(허브) 페이지입니다.  
프레임: **구조 >> 교차원리, 교차원리 기반 무도**

## 구조
- `index.html` : 메인 허브
- `assets/data/links.js` : 단체 목록(여기 URL/로고/설명만 수정하면 자동 반영)
- `assets/css/style.css` : 공통 스타일
- `assets/js/app.js` : 오비트 회전/정지, 모달, 링크 복사/공유

## 사용법
1) `assets/data/links.js`에서 각 단체의 `url`을 실제 주소로 변경  
2) `assets/images/`에 로고 파일을 넣으면 자동 표시됩니다  
3) GitHub Pages 활성화 후 접속

## 메모
- 오비트에서 마우스 올리면 회전 정지
- 클릭하면 정보/공유 모달 오픈
